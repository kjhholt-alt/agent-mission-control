# Report Templates

## Monthly Close Report Structure
1. **Executive Summary** (1 paragraph)
   - Net income vs budget, key driver
   - Biggest positive and negative variance
   - Action items requiring management attention

2. **Revenue Summary**
   - Revenue by product/segment
   - MTD and YTD comparisons
   - Trend vs prior year

3. **Expense Analysis**
   - Operating expenses by category
   - Variance vs budget with explanations
   - Headcount and compensation costs

4. **Key Variances** (table format)
   | Category | Budget | Actual | Variance $ | Variance % | Root Cause |

5. **Balance Sheet Highlights**
   - Cash position
   - AR/AP aging summary
   - Any unusual items

6. **Outlook & Recommendations**
   - Forecast for remainder of quarter
   - Risks and mitigation
   - Action items with owners and deadlines

## Variance Analysis Template
For each material variance:
- **Line Item**: [name]
- **Budget**: $X
- **Actual**: $X
- **Variance**: $X (X.X%)
- **Classification**: Timing | Volume | Rate | One-time | Structural
- **Root Cause**: [explanation]
- **Action Required**: Yes/No — [details if yes]

## Email Report Format
- Subject: [Period] Financial Summary — [Key Takeaway]
- Lead with the conclusion, not the data
- Tables for detailed numbers
- Bold key figures
- End with clear next steps / action items

## Presentation Slide Format
- Title slide: Report name, period, date
- Summary slide: 3–5 bullet points, key metrics
- Detail slides: one topic per slide, 3–5 points max
- Chart slides: single chart with clear takeaway in title
- Recommendation slide: numbered action items
- Appendix: detailed tables, supporting data
