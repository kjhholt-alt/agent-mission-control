## Summary

I've created a comprehensive **side-by-side scenario comparison** feature for the PL Engine that shows all 5 scenarios (Super Trough/Trough/Normal/Peak/Super Peak) in one view.

### What It Does

**Terminal Output (Rich Table):**
- Side-by-side values for all 5 scenarios
- Deltas from Normal for ST and SP (color-coded: green = lower cost, red = higher)
- Percent changes from Normal
- **Top 3 Biggest Movers** by absolute $ and by %
- Warning for held-constant categories

**Excel Output:**
- Professional formatted workbook with:
  - All scenario values in columns
  - Delta calculations (ST and SP vs Normal)
  - Conditional formatting (green = favorable, red = unfavorable)
  - **Highlighted biggest movers** (yellow background)
  - Summary section with metadata
  - Top movers table

### How to Use

```bash
# Generate comparison for one factory
python run.py --compare --factory AX02

# Output: terminal table + outputs/AX02_scenario_comparison.xlsx
```
