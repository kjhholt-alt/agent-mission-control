## Usage

```bash
# Parallel (new default, ~2 min)
python run.py --all

# Sequential (original behavior, ~10 min)
python run.py --all --sequential

# Single factory (still sequential)
python run.py --factory AX02
```

The parallel implementation uses as many workers as there are factories (5 max), and the org packet building still happens after all factories complete. Each worker process gets its own logger setup to avoid conflicts.
