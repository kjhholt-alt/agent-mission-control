## Summary of Changes

Enhanced `bot/watchdog.py` with the following features:

### 1. **Discord Webhook Retry Logic**
- 3 retry attempts with 2-second delay between failures
- Detailed logging of retry attempts and final failures
- Returns success/failure status for downstream handling

### 2. **New Monitoring Checks**
- **Balance monitoring**: Alerts when liquid USDC falls below `MIN_WALLET_HARD_FLOOR` ($35)
- **Position limit tracking**: Alerts when whale positions reach `WHALE_MAX_POSITIONS` (8)
- **Stricter staleness**: Reduced from 45min to **15min** threshold for trade activity
- **Error detection**: Enhanced Railway log scanning for critical errors

### 3. **Daily Summary Report (8am UTC)**
- 24-hour P&L and trade statistics
- Win/loss record and win rate
- Current balance and total portfolio value
- Top 5 strategies by performance
- Auto-sends when run between 8:00-8:15 UTC
- Manual trigger via `--daily-summary` flag

### 4. **Enhanced Discord Embeds**
- Added balance, position count, and cycle info to alerts
- Color-coded embeds (red for critical, orange for warnings, green for summaries)
- Detailed field breakdowns for quick diagnosis
