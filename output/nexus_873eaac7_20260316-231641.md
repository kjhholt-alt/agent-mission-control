✅ **Performance monitoring dashboard complete!**

## What I Built:

### 1. **Performance Page** (`/performance`)
- Real-time metrics dashboard with 30-second auto-refresh
- Time window selector: 1h, 6h, 24h, 7d
- Key metrics cards with visual indicators
- Queue depth chart over time
- Comprehensive performance summary

### 2. **API Endpoint** (`/api/performance`)
- Aggregates data from `swarm_tasks` and `swarm_workers` tables
- Calculates:
  - **Execution times**: average and median
  - **Error rate**: failed tasks / total tasks
  - **Success rate**: completed tasks / total tasks
  - **Worker utilization**: active workers / total workers
  - **Queue depth**: hourly bucketed data

### 3. **Trend Indicators**
- Compares first half vs second half of time window
- Shows ↑/↓ arrows with percentage change
- Red for increasing errors, green for decreasing

### 4. **Alerting Thresholds**
- 🔴 **Error rate > 5%** → Error alert
- ⚠️ **Avg execution > 300s** → Warning alert
- ⚠️ **Queue depth > 20** → Warning alert

### 5. **Metrics Tracked**
- Total tasks, completed, failed, running, queued
- Error rate & success rate
- Average & median execution time
- Worker utilization (active/total)
- Queue depth over time (hourly buckets)
