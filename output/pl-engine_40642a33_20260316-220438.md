## Summary

I've added comprehensive error handling to the PL Engine pipeline with meaningful error messages and actionable suggestions. Here's what was done:

### 🎯 What Was Changed

**1. New Error Infrastructure (`src/pipeline.py`)**
- Created `PipelineStageError` exception class that captures:
  - Which factory and stage failed
  - What input data was being processed
  - Original exception with full traceback
  - Context dict (file paths, config values, data samples)
  - Actionable fix suggestions
  
- Created `safe_stage_execution()` wrapper that handles:
  - FileNotFoundError → Shows expected paths, network share checks
  - PermissionError → Suggests closing Excel, permission fixes
  - ValueError → Config validation help
  - KeyError → Missing column/field details
  - ImportError → pip install suggestions

**2. Enhanced Pipeline Stages (`scripts/run_full_pipeline.py`)**

Enhanced **step0 (Pre-Export Validation)**:
- ✅ Success: Shows CSV format, row count, cost centers
- ❌ Errors: Missing columns, config issues, adjustment problems
- 💡 Suggestions: Expected formats, common fixes

Enhanced **step1 (Process Templates)**:
- ✅ Success: Shows script output, file sizes, directory checks
- ❌ Errors: Missing templates, script failures, permission issues
- 💡 Suggestions: Directory structure, common causes

Enhanced **step2 (Build Dashboard)**:
- ✅ Success: Shows budget totals, segment breakdowns, file creation
- ❌ Errors: Missing files, zero totals, locked files
- ❌ Warnings: Small files (<10KB), missing segments
- 💡 Suggestions: Memory issues, file locking, permissions

### 📊 Error Message Example

**Before**: Generic Python traceback
```
KeyError: 'scenario'
```

**After**: Rich context with suggestions

```
======================================================================
PIPELINE STAGE FAILURE
======================================================================
Factory:    AX02
Stage:      Step 2: Build Dashboard
Error Type: KeyError

Details: Missing required data field: scenario

Context:
  csv_path: 'C:\Network\AX02 Budget.csv'
  format: 'wide'
  missing_key: 'scenario'

Suggested Fix:
  1. Check that input CSV files have the expected column headers
  2. Expected columns: segment, cc, account, scenario, dollars
  3. For 'wide' format: must have CC, Account Name, then scenario columns
======================================================================
```
